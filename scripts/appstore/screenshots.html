How to export these as App Store screenshots:

Option A — Browser screenshot (quickest):
Open this file in Chrome. Right-click each phone frame → Inspect → select the .marketing-card element → right-click the node → "Capture node screenshot".

Option B — Automated with Puppeteer:
Run the companion script scripts/appstore/capture-screenshots.sh which uses Puppeteer to export each screenshot at exact App Store dimensions.

Option C — Design tool:
Use these as reference mockups. Recreate in Figma/Sketch at 1290×2796 with your actual simulator screenshots layered in.
