Automation

Run Sunclub from Shortcuts and URL actions.

Sunclub supports Apple Shortcuts, Control Center, widgets, custom URL scheme actions, and x-callback-url for supported non-destructive app actions.

Settings

Open Sunclub Settings -> Automation to manage Shortcut writes, URL open actions, URL write actions, and detailed callback results. Turn off URL writes if links from other apps should only open Sunclub screens.

Apple Shortcuts

  • Log Sunscreen with optional SPF and notes.
  • Save Sunscreen Log for today or a selected date and time.
  • Log Reapply.
  • Get Sunclub Status.
  • Open Sunclub to a supported screen.
  • Set reminder times and supported toggles.
  • Export a backup, create a skin health report, or create a streak card.
  • Import a friend invite or poke a friend.

URL Actions

Production links use sunclub://automation/.... Development builds use sunclub-dev://automation/.... Existing widget and accountability links continue to work.

  • sunclub://automation/log-today?spf=50&notes=Beach
  • sunclub://automation/save-log?date=2026-04-13&time=08:30&spf=50
  • sunclub://automation/reapply
  • sunclub://automation/status
  • sunclub://automation/set-reminder?kind=weekday&time=08:30
  • sunclub://automation/set-toggle?name=dailyUVBriefing&enabled=true
  • sunclub://automation/open?route=automation

x-callback-url

x-callback-url links use sunclub://x-callback-url/.... Success callbacks receive action, status, message, and action-specific fields when callback details are enabled. Error callbacks receive action, errorCode, and errorMessage.

If callback result details are off, callbacks only include action and status.

Foreground-Only Actions

Delete log, remove friend, backup import, recovery undo or redo, conflict resolution, camera scanning, file picking, and permission-only setup open Sunclub instead of writing in the background.

Privacy

URL actions can be called by other apps on your device. Disable URL writes in Sunclub Settings -> Automation if background links should not log sunscreen, reapply, update reminders, change supported toggles, import friends, or poke friends.
