Support

Sunclub support

For app issues, feedback, or feature requests, email support@mail.sunclub.peyton.app. Privacy questions go to privacy@mail.sunclub.peyton.app.

Logging today

Open Sunclub and use manual logging from Home. If today is already logged, the app keeps one record for the day and lets your progress stay consistent.

Reminders

Check notification permission in Apple system settings, then review daily, reapply, elevated UV, and weekly reminder settings in Sunclub.

Live UV and location

Live UV is optional and off by default. When you enable it, Sunclub may use your location to request local UV forecasts from Apple Weather through WeatherKit. Manual logging continues to work if location, network access, or Apple Weather is unavailable.

Widgets and controls

If widget state looks stale, open Sunclub once so the shared snapshot can refresh, then return to the widget or Control Center action.

Shortcuts and URL actions

Automation settings control Shortcut writes, URL opens, URL writes, and x-callback-url detail fields. The automation docs include examples and privacy controls.

Backup and iCloud sync

Local backup export/import and private iCloud sync live in Settings. Imported backups can be reviewed before you publish imported changes to iCloud.

Before emailing support

  • Include your iPhone model and iOS version.
  • Describe what you expected and what happened instead.
  • For reminder issues, note whether notifications are allowed in Apple system settings.
  • For sync or backup issues, note whether iCloud sync is paused in Sunclub settings.
  • For Shortcut or URL issues, include the action name and whether URL writes are enabled in Settings -> Automation.

Documentation

The docs collect setup notes for logging, reminders, widgets, automation, private sync, backup, and privacy. For automation examples, including friend invite import and message-nudge routes, read Automation Docs.

Security

For security reports, email security@mail.sunclub.peyton.app.

Privacy

Sunclub has no app-owned accounts, no ads, and no third-party analytics SDKs. Read the privacy policy for details about local storage, optional private iCloud sync, notifications, location, HealthKit, camera/product scanning, and backups.

General Contact

General questions can go to contact@mail.sunclub.peyton.app.

Medical Disclaimer

Sunclub is a habit tracker, not a medical device or medical advice. Follow sunscreen product directions and consult a qualified professional for medical questions.
